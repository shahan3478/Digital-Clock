# Digital Clock

This project is a clean and responsive **Digital Clock** application built using vanilla web technologies. It displays the current local time in a sleek, modern interface, updating in real-time to provide a seamless user experience.

---

## 🚀 Features

*   **Real-Time Updates**: Automatically updates the time every second using JavaScript’s `setInterval` function.
*   **Responsive Design**: The layout adapts to various screen sizes, ensuring the clock looks great on desktops, tablets, and mobile devices.
*   **Clean UI**: Minimalist aesthetic with a focus on readability and modern typography.
*   **Vanilla Implementation**: Built without external libraries or frameworks for maximum performance and a lightweight footprint.

---

## 🛠️ Project Structure

The project consists of the following core files located within the `Digital-Clock-main` directory[cite: 1]:

*   **`index.html`**: Defines the semantic structure and layout of the clock[cite: 1].
*   **`style.css`**: Contains the styling, including layout positioning, colors, and responsive media queries[cite: 1].
*   **`script.js`**: Houses the logic for fetching the system time and dynamically updating the DOM[cite: 1].



---

## 💻 Tech Stack

*   **HTML5**: Structure and content.
*   **CSS3**: Flexbox/Grid layout and custom styling.
*   **JavaScript (ES6+)**: Time calculations and DOM manipulation.

## ⚙️ Setup & Usage

To run this project locally, simply follow these steps:

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/your-username/Digital-Clock.git
    ```
2.  **Navigate to the directory**:
    ```bash
    cd Digital-Clock-main
    ```
3.  **Open the project**:
    Open `index.html` in any modern web browser to view the clock[cite: 1].
```
